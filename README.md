# Machine Learning Learning & Portfolio Repository

Welcome to my Machine Learning repository! This project serves as a comprehensive log of my journey through ML, tracking my progress from foundational algorithms to practical data-driven solutions. It contains a mix of core concept implementations and real-world project notebooks.

## Repository Structure

### Core Concepts & Algorithms
**01- Introduction to ML.ipynb**
    * An introduction to the fundamentals of machine learning, covering core workflows, data preprocessing, and the difference between supervised and unsupervised learning.
**02- Linear Regression, Ridges, Lasso, and Logistic Regression.ipynb**
    * Deep dive into regression techniques, regularization methods to prevent overfitting (L1/L2 penalties), and fundamental binary classification.
**03- Decision Tree Classifier.ipynb**
    * Implementation of tree-based models, understanding split criteria (Gini impurity, Information Gain), and visualizing decision boundaries.
**04- Clustering.ipynb**
    * Exploring unsupervised learning principles, distance metrics, and the mathematical foundations of grouping unlabeled data.

### Applied Projects & Implementations
**05- House Price Prediction**
    * Algorithm: Linear Regression, Ridge, and Lasso.
    * Goal: Predicting real estate prices based on housing features through feature engineering and regularized regression modeling.
**06- Diabetes Prediction**
    * Algorithm: Logistic Regression & Classification Techniques.
    * Goal: A healthcare-focused classification project aimed at predicting diagnostic outcomes based on clinical metrics.
**07- Weather Data Prediction**
    * Algorithm: Random Forest
    * Goal: Analyzing historical climate features to model and predict upcoming weather patterns and trends.
**08- Customer Segmentation**
    * Algorithm: K-Means Clustering.
    * Goal: Segmenting e-commerce or retail client bases into distinct behavioral groups to optimize targeted marketing strategies.
**09- Spam Email Classifier**
    * Algorithm: Naive Bayes.
    * Goal:* A Natural Language Processing (NLP) text classification system that filters incoming messages into spam or ham using probabilistic modeling.

## Tech Stack & Tools
**Language:** Python
**Libraries:** NumPy, Pandas, Scikit-Learn, Matplotlib, Seaborn
**Environment:** Jupyter Notebooks / VS Code

* **Problem Solving:** Structured data workflows to address both supervised (regression/classification) and unsupervised (clustering) challenges.
